READ ME:

REMOVE PREVIOUS OFFICE:

-Search "Add or Remove Programs"

-Search for "Office" and install

-Search for "One Note" and install


INSTALLATION:

-Copy this folder onto the desktop of the Admin account on this computer

-Open the folder

-Double click Install file

-When Pop up appears asking to allow Microsoft to make changes click "Yes"


ACTIVATION: 

To activate the office apps open Microsoft Word (search the computer for it)

On the Bottom Left click "Account"

Under "Product Information" click "Change Product Key" > enter provided key

Now log into guest account and it will prompt for activating, follow the wizard and choose internet.
